Opis programu:
Program wylicza wartosci odwzorowania logistycznego dla k w zakresie [2;4] i wypisuje je do pliku "dane.dat".


Kompilacja i uruchomienie:
Program korzysta z biblioteki matematycznej, wiec nalezy kompilowac z flaga -lm.
gcc main.c -o main
./main

Rysowanie wykresu z danych:
Aby z danych wypisanych przez program do pliku "dane.dat" stworzyc wykres w programie gnuplot nalezy wpisac polecenie:
gnuplot wykres.gp
