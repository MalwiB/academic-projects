Program oblicza dla danego N wartosci zadanej sumy metoda uzywajaca w kazdej iteracji funkcji z biblioteki matematycznej- cos() i exp() oraz za pomoca funkcji zoptymalizowanej uzywajacej tych cos() i exp() tylko raz. Istnieje okreslona dokladnosc wyniku.

Nalezy kompilowac Z FLAGA -lm (biblioteka matematyczna).
