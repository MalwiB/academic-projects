Opis programu:
Program znajduje dla podanej funkcji przedzialy w ktorych znajduja sie miejsca zerowe, a nastepnie znajduje w tych przedzialach miejsce zerowe metodami:
* bisekcji
* regula-falsi
* siecznych
* Newton'a
* Halley'a

Kompilacja i uruchomienie:
Program korzysta z biblioteki matematycznej C, nalezy kompilowac z flaga -lm.

gcc main.c -o main -lm
./main
