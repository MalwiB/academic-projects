--Jak uruchomic program:
Aby uruchomić program pod systemem Linux, najpierw należy go skompilować, wpisując polecenie make albo make all, a nastepnie go uruchomic poleceniem "make run", ewentualnie wpisujac w terminalu "./bakery_algorithm.x".


Dodatkową opcją jest możliwość utworzenia z plików archiwum za pomocą polecenia "make tar" oraz wyczyszczenia powstałych przy linkowaniu oraz kompilowaniu programów plików obiektowych i wykonywalnych poprzez "make clean".
